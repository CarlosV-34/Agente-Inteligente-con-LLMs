# 🧠 Agente Inteligente con LLMs

## 📌 Descripción

Implementaremos un agente basado en modelos de lenguaje (LLMs) capaz de realizar tareas de procesamiento de texto como resumen, análisis de sentimiento, extracción de información y generación de contenido.

---

## 🎯 Objetivos

En este proyecto trabajaremos con modelos de lenguaje preentrenados para:

* Cargar y ejecutar un LLM en local
* Generar texto a partir de prompts
* Ajustar parámetros de generación (temperature, top-k, top-p)
* Evaluar distintas estrategias de uso del modelo
* Construir un pipeline de análisis de texto

---

## ⚙️ Tecnologías

Utilizaremos las siguientes herramientas:

* Python
* PyTorch
* Transformers (Hugging Face)
* Jupyter Notebook

---

## 🧪 Funcionalidades

El agente será capaz de:

* Detectar el idioma de un texto
* Generar resúmenes
* Extraer palabras clave
* Realizar análisis de sentimiento
* Generar preguntas a partir del contenido
